DiNAR App
=========

The `DiNAR <https://nib-si.shinyapps.io/DiNAR/>`_ (Differential Network Analysis in R) app is a tool for generating and analysing dynamic biological networks. For a more detailed use of DiNAR as an individual tool, access the following `tutorial <http://conferences.nib.si/DiNAR/>`_. 
The code for DiNAR is publicly available on the `DiNAR github page <https://github.com/NIB-SI/DiNAR>`_.
